100均ペンプロッタ ツール集

iPhone / PC から使える、micro:bit ペンプロッタ用のデータ作成＆編集ツールの入り口ページです。

① お絵描き＆RAW5生成 メイン

指やマウスで絵を描き、0–180 グリッドにスナップして RAW5 / JSON を生成。下絵の読み込み・スムージング・間引き対応。

  • 手描き → RAW5 / JSON 出力
  • SVG / JSON / RAW5 読み込み
  • 下絵の背景表示（濃さ調整）
  • ズーム / パン / スムージング
🖊️お絵描きツールを開く（draw.html）

② プロッタ画像・合成エディタ

既存のプロッタデータ（RAW5 / JSON）を複数レイヤとして読み込み、拡大縮小・平行移動して合成するためのツール。

  • 複数レイヤの重ね合わせ
  • レイヤごとのスケール・移動調整
  • 合成結果を RAW5 / JSON に書き出し
🧩合成エディタを開く

③ 2枚比較ビューワー

2つのプロッタデータを左右並列・重ね・点滅表示で比較するツール。調整前後の差分確認に便利。

  • 左右並列 / 重ね / A/B点滅モード
  • ズーム / パンで詳細確認
  • 全体距離や点数の簡易メトリクス表示
🔍2枚比較ツールを開く

④ フレーム密度 ×2 ツール

RAW5 / JSON のフレーム間に中間点を挿入して、見かけ上の描画解像度を 2倍にするユーティリティ。

  • RAW5 / JSON を読み込み
  • フレーム間に中間点を挿入（×2）
  • 結果を新しいファイルとしてダウンロード
➕フレーム倍密度化ツールを開く

⑤ 4000文字ごと改行ツール

改行のないテキストを 4000 文字ごとに区切って 2連改行を挿入する整形ツール。RAW5 テキストの分割にも利用可能。

  • 任意テキスト / ファイルを入力
  • 4000文字ごとに空行入りで整形
  • コピー / ダウンロード可能
✂️テキスト整形ツールを開く

⑥ micro:bit ペンプロッタ用プロジェクト

ペンプロッタ本体に取り付けた micro:bit に書き込むための MakeCode プロジェクト集。 基本操作用プロジェクトと、描画データ入りのお試しプロジェクトがあります。

⑥-1. 基本操作用プロジェクト

  • ペンプロッタ基本プロジェクト（操作用）
    ペンの上下・XY制御など、プロッタの基本動作を行う最小構成。
    📦 .hex をダウンロード

⑥-2. サンプル図形データ入りプロジェクト

  • サンプルA：○○の線画（データ入り）
    RAW5 データを内蔵したお試し用。書き込むだけですぐ描画できます。
    📦 サンプルA .hex ダウンロード
  • サンプルB：△△の図形（データ入り）
    別図形データのデモプロジェクト。動作確認用に便利。
    📦 サンプルB .hex ダウンロード
※ 各ツールは同一リポジトリ内の HTML を前提にしています。ファイル名を変更した場合は、リンク先（href）も合わせて修正してください。
