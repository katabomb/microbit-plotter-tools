RAW5 Viewer / Converter microbitペンプロッタ互換

RAW5正式仕様（PYYXX / 1フレーム5文字）をそのまま扱う単一HTMLアプリです。
空白・改行を無視して読み込み、連結RAW5 / 20フレームごと改行RAW5 を書き出せます。

入力RAW5

フレーム数
0
ストローク数
0
penUp数
0
penDown数
0
読み込み時は空白・改行を無視し、5文字ずつ切り出して解釈します。
1フレームは PYYXX で、P=1 は penUp、P=0 は penDown です。

プレビュー

座標範囲は RAW5仕様に従い x,y = 0〜180 です。

RAW5書き出し

連結RAW5（空白なし・改行なし）
20フレームごと改行RAW5（フレーム間スペースあり）

関数テスト

仕様上必要な関数 packFrame / unpackFrame / parseRAW5 / framesToRAW5 を実装済みです。
